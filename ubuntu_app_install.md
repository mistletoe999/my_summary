##Install softwares via "Terminal"
* R
```
sudo apt-get install r-base-core
```
* vim
```
sudo apt-get install vim
```
* Latex
```
sudo apt-get install texlive
sudo apt-get install texmaker
```
Online tutorial: http://www.openvim.com/tutorial.html
