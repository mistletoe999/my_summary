

##Naming Conventions
| type|     name |
|---------|------|
|   directory/package  |  dir_name    | 
|  file/module|file_name|
|namespace|space_name|
| class | ClassName|
|function|funName|
|variable|var_name|
|constant/macro|CONST_NAME|
